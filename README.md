WarpWrap - VS Code Extension
WarpWrap is a powerful VS Code extension that lets you effortlessly wrap HTML elements with custom tags, class names, and IDs. Featuring smart indentation, live preview, and undo support, it streamlines your coding workflow for faster and cleaner HTML structuring.

Features
Wrap with Tag: Quickly wrap selected HTML elements with any tag.
Wrap with Config: Customize wrapping with predefined configurations.
Smart Wrap: Intelligently applies the most relevant wrapping based on context.
Live Preview: Instantly preview wrapped elements within VS Code.
Extract Inline Styles: Convert inline styles to separate CSS classes.
Undo Last Wrap: Revert the last applied wrap with one click.
Installation
Open VS Code.
Go to the Extensions Marketplace (Ctrl+Shift+X).
Search for WarpWrap.
Click Install.
Usage
Open an HTML file in VS Code.
Select an element or multiple lines of code.
Open the Command Palette (Ctrl+Shift+P) and search for WarpWrap commands.
Configuration
WarpWrap uses a configuration file (warpwrap.json) for custom wrapping rules. You can specify custom class names, IDs, and other attributes.

Requirements
VS Code version 1.97.0 or higher.
